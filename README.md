# UEProject
練習用的Roguelike 3D射擊遊戲。
## 專案說明
* 遊戲引擎：Unreal Engine 5.7.4
* 支援平台：Windows
## 遊戲玩法
靠著手槍殲滅一波又一波的敵人。  
每次通關從隨機兩個能力中二選一：  
* 提升射速50%
* 提升傷害50%
* 提升彈藥容量100%
* 提升血量上限100
* 提升移動速度50%
